My Favorite HTML Tags

By: Chris Stewart

Introduction After doing some research I have found three Hypertext Markup Language(HTML) tags that are my favorite. I have used all three in this web page.

<hr> The horizontal rule tag is one feature I found to be very fun to use and helpful. It helps break up blocks of text and adds some color to web page making it easier to follow and also makes it more aesthetically pleasing.

<img> The image tag is one I enjoy very much. It allows you to add more visual content that makes a webpage easier to look at. It can catch the eye of the reader and allows you to show a picture to get your point across versus just text.

<h1> The header tags add a level of organization and order to a web page. It makes it easier to understand what areas of text or grouped together. If all the text on a web page was the same size and style it would be unreadable and unorganized.

In Conclusion, I appreciate the creative things like the Horizontal Rule tag, the Image tag, and the Header tag that can be used to make a simple HTML web page much more user friendly and visually pleasing.
