Twenty CSS Properties

By: Chris Stewart

Introduction

The purpose of this webpage is to show my favorite properties of CSS. CSS properties are very useful in making a webpage easier to read and follow. Otherwise it would just be words on the screen and have no formatting or colors to break it up.

Top Five CSS Properties

Shadow

1. This is a property I found quite fascinating. It can really spice up the styling of a Title or a Box so it looks like it is jumping of the page. It can be a attention getter and make a page easier to follow.

This property can be used as a text shadow or a box shadow.

Border

2. This is a neat property that can really emphasize a group of words to make them stand out.

This property can be used by adding a border-color, a border-width, a border-style, a border-image, a border-radius, a border-top, a border-left, a border-right, a border-bottom as well as many other combinations.

Spacing

3. This is a property that is nice to use when you want to change where on a screen an image or a block of text appear.

This property can be used by adding padding, padding-bottom, padding-left, padding-right, padding-top, margin, margin-bottom, margin-left, margin-right, or margin-top.

Animation

4. This is a nice way to catch the eye of your reader. It draws your attention to the screen and makes it come to life. It is something that can be overdone, which makes the webpage hard to read and follow. It is good to use but in moderation.

This property can be used in many forms of animation or transitions.

Layout

5. This property helps adjust the display of the webpage. It can help keep things on the screen when the screen size is different and it can change where a line of text starts. This property is very handy when you are using multiple columns.

This property can be used by adjusting the bottom, left, right, top of the webpage. It can also be used by changing the float or position of the text.
