# Bootstrap Class Reason for use
1 Navbar I used the navbar to create a menu to reach all the webpages.
2 Navbar fixed-bottom I used the fixed bottom feature to create a border at the botton of the page
3 Container-fluid I used the container-fluid to creat a container to place images and text that could resize as needed
4 navbar-inverse I used the inverse feature to create a different look on the menu bar
5 embed responsive item I used the embed-responsive feature to show a pdf within the webpage
6 text-*-Center I used the text-center to center text on the screen making it easier to read.
7 container I used a container to hold and display images and text on the screen. This allowed a pdf to be readable on any size screen.
8 image-responsive I used the image-responsive feature to allow the image to flex with the screen size
9 list group item I used the list group item class to creat a easy to read list of items. This class helped to break up the content.
10 form-group row I used for group row class to add a comment section to the page. Allowing someone to comment using their name, email, and comment.
11 ftable table-hover I used the table hover class to spice up a list.

Pros and Cons

There are many pros and cons to each woord press and bootstrap. I feel that for the beginner word press is a easier process in building a website. You can see the results faster and cleaner.

However bootstrap is more customizable for someone who knows what thay are doing. It takes a lot more work but can have the exact look on the screen you wanted. Having used both now in a few weeks time I can say I like the finished product of Wordpress but I like knowing I created the whole thing with bootstrap.
